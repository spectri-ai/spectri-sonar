# [ISSUES/] ISSUES FOLDER
<!-- target: spectri/issues/ -->

## Organization

Issues are dated markdown files in flat structure (e.g., `2026-01-19-description.md`).

Supporting folders:
- `resolved/` - Completed issues
- `screenshots/` - Issue attachments

## Issue Management

When you resolve an issue, run `resolve-issue.sh`. To reopen a resolved issue, run `reopen-issue.sh`. These scripts handle metadata updates and folder movement.
